# BubbleRunner

iOS Bubble Game

## Technologies

- iOS 10.3
- Swift 3
- SpriteKit
- Firebase Analytics
- Firebase AdMob

## Dependencies

BubbleRunner uses **Cocoapods** to handle dependencies. After cloning the repository please execute the following command in the terminal:

```bash
pod install
```
